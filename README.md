# 🌟 SmartService AI Platform — Frontend

> A modern Angular application for browsing, booking services, and viewing AI-generated review summaries.

---

## 📸 Overview

SmartService is a smart service marketplace where users can:
- Browse available services
- Book a service from a provider
- View AI-generated summaries of customer reviews (powered by **Groq + Llama 3.3 70B**)
- Authenticate securely with JWT

---

## 🚀 Tech Stack

| Technology | Usage |
|---|---|
| **Angular** | Frontend framework |
| **TypeScript** | Programming language |
| **HTML / SCSS** | UI & styling |
| **JWT** | Authentication |
| **REST API** | Communication with backend |

---

## ✨ Features

- 🔐 Login & Register with JWT authentication
- 📦 Browse and search services by category
- 📅 Book a service from a provider
- ⭐ Leave reviews and ratings
- 🤖 AI-powered review summarization (Groq API)
- 📱 Responsive design

---

## ⚙️ Installation & Setup

# 1. Clone the repository
git clone https://github.com/souhir-fazzaini/SmartService-AI-Platform-Front.git

# 2. Navigate to the project
cd SmartService-AI-Platform-Front

# 3. Install dependencies
npm install

# 4. Run the development server
ng serve

Then open your browser at:
http://localhost:4200

---

## 🔗 Backend

This frontend connects to the SmartService Backend:
👉 https://github.com/souhir-fazzaini/SmartService-Backend

Make sure the backend is running on http://localhost:8080 before starting the frontend.
